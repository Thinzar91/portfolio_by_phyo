ポートフォリオサイト — Webデザイン制作
自身の制作実績やスキルを整理・可視化するWebサイトを制作しました。
見やすく分かりやすいデザインと操作性を意識し、閲覧者が直感的に情報を理解できる構成にしています。
Webディレクション、デザイン、コーディングを担当し、PC／スマホ対応まで一貫して制作しました。
目的
自身のWebデザインスキルを体系的にまとめ、制作実績を効果的にアピールするためのポートフォリオサイトを制作しました。デザインの一貫性やユーザー体験を重視し、訪問者が直感的に操作できる構成を心がけました。
ターゲット
就職活動中の企業担当者や採用担当者、Webデザインのスキルや制作実績を確認したいクライアントを主なターゲットとしています。
コンセプト
自身の制作実績を整理・可視化し、閲覧者が直感的にスキルや経験を理解できる「見やすく分かりやすいポートフォリオサイト」を目指しました。 統一感のあるデザインと操作性を重視し、ブランドイメージの向上と自己PRの効果を両立しています。
デザインポイント
【カラー】：ホワイトとグレージュを基調に、落ち着いたトーンで全体をまとめました。シンプルで清潔感のある配色により、制作実績やコンテンツが引き立つデザインに仕上げています。

【レイアウト】：余白を効果的に使い、各セクションを整理して配置。訪問者が直感的に情報を探せるよう、視線の流れを意識した構成にしました。

【タイポグラフィ】：本文は読みやすさを重視したサンセリフ体を採用。見出しでは洗練されたフォントを使用し、ポートフォリオ全体にメリハリと統一感を与えました。

【写真構成】：制作実績のスクリーンショットを大きく配置し、各作品の特徴や質感が直感的に伝わるよう工夫しました。

【導線設計】：ナビゲーションやセクションリンクを分かりやすく配置し、訪問者がスムーズに各コンテンツにアクセスできるUI設計を意識しました。
担当範囲
・Webディレクション
・デザイン構成・ワイヤーフレーム作成
・ビジュアルデザイン（PC／スマホ対応）
・画像素材編集
・コーディング（HTML／CSS／JavaScript）
使用ツール
Figma / Photoshop / Illustrator / Canva / VS Code / HTML / CSS / JavaScript
制作期間
約3か月（企画・デザイン・コーディング・レスポンシブ対応含む）
